# ModRobloxX
Это чит на Roblox он содержит
Noclip - не получаешь урон в любых плейсах
AddRobux - добавление робуксов это не визул все это было взломанно
ESP - обычное esp которая показывает игроков через стенки
AIM - эта функция работает тоько в плейсах о стрельбе и шутарах плейсах

# Устоновка
1. Нажмите кнопку скачать на сайте
2. Скачайте инсталлер в виде .zip
3. Откройте файл RobloxCheat.exe.lnk это ярлык поэтому такое расширение
4. Играйте и наслаждайтесь
